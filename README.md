
# 🌍 Global Airbnb Performance Dashboard

## 📊 Project Overview
This project is a **Power BI dashboard** that analyzes global Airbnb performance across multiple years.  
It was created as a **learning project** to understand and implement basic Power BI tools, including data visualization, KPI cards, and trend analysis.

---

## 📁 Dataset
This project uses a dataset from Maven Analytics:

👉 https://mavenanalytics.io/guided-projects/airbnb-listing-analysis

---

## 🎯 Objectives
- Analyze Airbnb listing growth over time  
- Compare different room types (Entire, Private, Shared, Hotel)  
- Understand key metrics like listings, hosts, reviews, and cities  
- Practice Power BI dashboard creation  

---

## 📈 Dashboard Features
- KPI Cards (Listings, Cities, Hosts, Property Types, Reviews)  
- Year-wise trend analysis of listings  
- Room type comparison  
- Clean and interactive dashboard design  

---

## 🛠️ Tools Used
- Power BI  
- Data Visualization  
- Data Cleaning  

---

## 📊 Key Insights
- Rapid growth in Airbnb listings between 2012–2016  
- Entire homes dominate the market  
- Private rooms are the second most common listing type  
- Listings show decline after peak years
